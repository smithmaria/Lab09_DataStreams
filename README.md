# File Search Processor
This is a Java Swing program that will load and search a text file by a keyword.

## How to Use
1. Click "Load File" and select a text file
2. Enter a keyword in the "Search String" input box.
3. Click "Search File" to search for the keyword in your file.
   All lines containing that keyword will appear in the "Filtered Results" textarea on the right side of the window. The original file will appear on the left side.
